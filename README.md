# Library Management Website

A library management system built with a ReactJS frontend and an ExpressJS backend. This project allows you to manage books, users, and loan transactions with role-based access control. There are two main roles: **Admin** and **User**.

## Features

- **User Authentication & Authorization**
  - Secure login system with role-based access.
  - **Sample Admin Account:**
    - **Email:** `admin@library.com`
    - **Password:** `adminpass`

- **Book Management**
  - Add, update, delete, and view books.
  - Manage book loans and returns.

- **User Management**
  - Register and manage user profiles.
  - Role assignment for Admins and regular Users.

- **Responsive Design**
  - Built with ReactJS and Tailwind CSS for a modern, responsive UI.

- **Backend API**
  - RESTful endpoints built with ExpressJS.
  - Secure and scalable server-side logic.

## Technology Stack

- **Frontend:** ReactJS, Tailwind CSS, Axios (for API calls)
- **Backend:** Node.js, ExpressJS, MySQL (or your preferred relational database)
- **Authentication:** JWT or session-based authentication
- **Deployment:** Vercel (frontend) and Heroku or other hosting solutions for the backend
